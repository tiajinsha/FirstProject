#基于vue脚手架的移动端聚合网站
=====
示例网站:http://tiajinsha.hkg03.bdysite.com
目录介绍
  1.src项目代码
    1.1 assets-静态文件
    1.2 components-子组件
    1.3 lib-mui
    1.4 Storage-模块(用于每次保存登陆成功账号)
    1.5 data.vue 全局api接口
  2.server后端代码
首页：
## 安装依赖
```
npm install
```
### 运行
```
npm run serve
```
### 打包
```
npm run build
```
